tomcat-any-ver Cookbook
=======================
Installs Apache Tomcat in any version e.g. 6.0.35 or 7.0.47


Requirements
------------
Tested on tomca6 and tomcat7

Attributes
----------
default['tomcat-any-ver']['version'] = "X.X.X" (defaults to 7.0.33)
